# 新疆伊犁3天2夜行程

## 行程参数
- **日期**：2026年5月27日（周三）至 2026年5月29日（周五）
- **出发地**：北京
- **同行人**：4人（两对情侣）
- **交通方式**：飞机往返 + 包车/自驾
- **预算**：经济舒适型（人均约¥3500-4500）
- **节奏**：轻松浪漫，适合情侣拍照打卡

| 19:00-20:00 | 返回酒店，稍作休整 | - | - | - |

**今日餐食**：
- 早餐：酒店含早
- 午餐：赛里木湖景区内（建议自备干粮和水，景区餐饮贵且选择少）
- 晚餐：伊宁市区本地餐厅

**休息与补给**：
- 赛里木湖海拔约2000米，注意保暖，备冲锋衣或薄羽绒服
- 湖边风大，建议携带防风外套和防晒用品
- 自备干粮和水，景区内餐饮选择少

**核心体验/首推项目**：
- 赛里木湖环湖自驾：随时停车拍照，欣赏雪山湖泊美景
- 果子沟大桥：壮观峡谷景观，中国最美公路桥
- 日落拍照：赛里木湖日落约20:00（新疆时间），光线柔和，适合拍照

**预算估算**：
- 包车：¥300-500/车/天（4人分摊约¥75-125/人）
- 赛里木湖门票：¥70-145/人
- 午餐（自备干粮）：¥30-50/人
- 晚餐：¥100-150/人
- **Day 2 小计**：约¥275-470/人

**预约提醒**：
- 赛里木湖旺季建议提前在OTA平台预约门票

**雨天/疲劳备用**：
- 若天气不佳，可改为伊宁市区深度游：伊犁河大桥、六星街、汉人街
- 若疲劳，可缩短赛里木湖游览时间，下午提前返回伊宁休息
