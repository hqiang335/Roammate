## 逐项分析

### 橘子洲景区
- **推荐理由**：湘江中的冲积沙洲，毛主席青年艺术雕塑所在地，免费开放，日落夜景绝佳
- **首推体验**：毛主席青年艺术雕塑打卡、百米高喷、湘江江景步道
- **建议时长**：2小时（白天）；若看夜景可延长至3小时
- **常见差评**：节假日人流极大，观光小火车排队长
- **避雷点**：不要购买黄牛票（景区有专用投诉邮箱，黄牛票无法入园）；观光小火车可提前在「龙骧橘子洲」小程序购票
- **适合**：所有人，穷游首选（免费）
- **不适合**：不喜欢人多的游客（节假日）
- **预约/排队/价格提醒**：必须提前3天在「岳麓山橘子洲旅游区」微信公众号预约，超时未入园需重新预约；开放时间07:00-22:00
- **来源信号**：Web-Rooter cited（搜狐官方预约攻略）+ FlyAI semantic reference

### 岳麓山国家重点风景名胜区（含爱晚亭）
- **推荐理由**：长沙地标，古迹众多，免费开放，5月绿意盎然，适合徒步
- **首推体验**：爱晚亭（免费）→ 古麓山寺 → 山顶观光长廊俯瞰长沙城
- **建议时长**：4-5小时（含岳麓书院）
- **常见差评**：山路较长，体力消耗大；节假日人多
- **避雷点**：5月多雨，建议穿防滑运动鞋；索道雨天可能停运；山上餐饮物价高，建议自备干粮
- **适合**：穷游徒步爱好者、体力好的年轻人
- **不适合**：老人、行动不便者（山路较陡）
- **预约/排队/价格提醒**：进山免费，需在「岳麓山橘子洲旅游区」公众号预约；可提前3天预约；地铁4号线「湖南大学」站2号口出步行可达南门
- **来源信号**：Web-Rooter cited（搜狐预约攻略）+ FlyAI semantic reference

### 岳麓书院
- **推荐理由**：中国古代四大书院之一，千年学府，建筑古朴，碑刻丰富
- **首推体验**：讲堂、御书楼、文庙、碑廊
- **建议时长**：2小时
- **常见差评**：需购票，部分游客觉得内容较学术
- **避雷点**：建议提前在官方公众号预约购票，避免现场排队
- **适合**：文化历史爱好者、穷游（¥40性价比高）
- **不适合**：只想拍照打卡不深入了解的游客
- **预约/排队/价格提醒**：门票¥40（学生¥20），需预约；地址：麓山路273号
- **来源信号**：FlyAI semantic reference

### 湖南博物院（马王堆汉墓）
- **推荐理由**：馆藏马王堆汉墓文物（素纱单衣、辛追夫人、彩绘帛画），免费开放，是长沙最顶级的文化地标
- **首推体验**：马王堆汉墓陈列（辛追夫人、素纱单衣）、湖南人·三湘历史文化陈列
- **建议时长**：3-4小时（深度游可达5小时）
- **常见差评**：票极难抢，周一闭馆
- **避雷点**：必须提前7天在「湖南博物院」微信公众号20:00准时抢票；若抢不到可考虑购买含代约服务的讲解套餐（FlyAI有售，价格约¥100-200含门票+讲解）；周一闭馆（法定节假日除外）
- **适合**：历史文化爱好者、穷游（免费）
- **不适合**：不喜欢博物馆的游客
- **预约/排队/价格提醒**：基本陈列免费，特展可能收费；地铁6号线「烈士公园南」站或1号线「文昌阁」站步行可达；旺季上午排队较长，建议预约最早时段
- **来源信号**：FlyAI semantic reference + Web-Rooter cited（百度移动搜索攻略）

### 太平老街/坡子街
- **推荐理由**：长沙最具代表性的历史街区，贾谊故居、长怀井等古迹，臭豆腐、糖油粑粑集中地
- **首推体验**：黑色经典臭豆腐、李公庙糖油粑粑、茶颜悦色（任意门店）
- **建议时长**：1-2小时（傍晚逛吃最佳）
- **常见差评**：主街商业化严重，价格偏高，部分小摊卫生一般
- **避雷点**：不要在主街不知名摊贩购买（价格虚高）；火宫殿名气大于口味，仅打卡不用餐；茶颜悦色不需要在总店排队，任意分店口味相同
- **适合**：美食打卡族、穷游（小吃人均30-50元）
- **不适合**：追求精致用餐环境的游客
- **预约/排队/价格提醒**：开放式街区，无需预约；地铁1号线「黄兴广场」站或「五一广场」站步行可达
- **来源信号**：FlyAI semantic reference

### 超级文和友（海信广场店）
- **推荐理由**：复古装修还原80年代老长沙社区场景，拍照极出片
- **首推体验**：免费参观内部场景拍照，感受氛围
- **建议时长**：1小时（仅拍照不用餐）
- **常见差评**：用餐排队极长（饭点可能等数百桌），消费较高
- **避雷点**：非饭点前往拍照打卡，不建议在此用餐；地铁1号线「南门口」站附近
- **适合**：打卡拍照族
- **不适合**：想在此用餐的穷游客（性价比低）
- **预约/排队/价格提醒**：可在小程序提前取号；地址：坡子街街道（海信广场）
- **来源信号**：FlyAI semantic reference

## 住宿区域策略

### 区域判断
**首选：五一广场/黄兴广场/芙蓉广场区域**
- 地铁1号线+2号线交汇，步行可达太平老街、坡子街、超级文和友
- 距橘子洲（地铁2号线）、岳麓山（地铁4号线换乘）均方便
- 夜生活丰富，便利店、小吃密集
